# Eventmanager app


# Generate UID's in attendees.yml, the invitee db
```
ruby addUID.rb
``` 

# Mail invitees
Make sure the mail server and mail_sender are defined in mail.rb

```
ruby mail.rb
```

# Running the app
You can change the default address + port in the eventmanager.rb file.
``` 
ruby eventmanager.rb
```
